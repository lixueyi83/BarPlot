# BarPlot
Plot bar chart with a single bash script, no any other dependencies. Handy, easy, even though a little bit dirty 😉

## Usage

```bash
    ./plot.sh 10    # 10 is the scalar, could be any number
```

- **Input:**

    `./data.txt`:

    ```csv
    ┌─────────────────┐
    │  Eli,2          │
    │  Job,0          │
    │  Joe,15         │
    │  Matt,4         │
    │  Act,305        │
    │  Alex,8         │
    │  David,100      │
    └─────────────────┘
    ```
